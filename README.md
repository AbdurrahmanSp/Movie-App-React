# Movie App

## About project

- Reusable components
- Reusable functions
- Folder structure (API, components, views) 
- Responsive view (Desktop, Tablet, Mobile)

### Built With
- ReactJS
- Styled Components
- The Movie Database API

## Available commands:

Install packages

```sh
npm install
```

Run project

```sh
npm start
```
